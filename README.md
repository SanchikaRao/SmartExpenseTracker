# 💳 FinTrack AI - Smart Expense Tracker with OCR

An intelligent full-stack expense tracking web application that uses Optical Character Recognition (OCR) to automatically extract, log, and categorize spending from receipt photos.

## 🚀 Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS, Lucide Icons, Recharts, React Router
- **OCR Engine:** Tesseract.js (Client-side text recognition)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose Schema)

## ✨ Features
- **Instant OCR Scanning:** Upload receipt photos to extract amounts and auto-assign categories (Food, Utilities, Academics, etc.).
- **Interactive Analytics:** Visual budget balance cards and spending distribution charts.
- **RESTful API:** Full CRUD integration connecting React frontend to Express/MongoDB backend.
- **Multi-Page Mobile UI:** Clean mobile-first design with interactive navigation routes.
